teapot - A simple graph library
====


How to add dependencies
----
```toml
### ssh
teapot = { git = "ssh://git@github.com:22/K-atc/teapot-rs.git" }
### https
teapot = { git = "https://github.com/K-atc/teapot-rs" }
```


How to build
----
```shell
cargo build
```


How to test
----
```shell
cargo test
```
